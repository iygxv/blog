## **代码格式自动整理**

```html
Shift + Option + F
```

## **查找与替换**

单文件查找：

```html
Cmd + F
```

所有文件查找：

```html
Cmd + Shift + F
```

同时选中多个相同单词：

```html
Cmd + D
```

## **复制粘贴**

快速复制粘贴一行

```html
Shift + Option + ↑ / ↓
```

## **光标定位**

定位到行首尾：

```html
Fn + ← / → 
```

定位到单词首尾：

```html
Option + ← / → 
```

添加多个光标：

```html
Cmd + Option + ↑ / ↓
```

选中一块区域：

```html
Shift  +  Option  +  拖动鼠标
```

移动本行代码
```html
Alt + ↑ / ↓
```

## **快速定位到某一行**
```html
 ctrl + g
```
