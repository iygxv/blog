## 一些常用的 linux 命令
### 查找指定文件夹
```shell
find . -name 'nginx'
```
. 表示当前目录，-name 指定文件名

### 在根目录查找文件夹
```shell
find / -name 'nginx'
```
### 查看某一进程
```shell
ps -ef | grep nginx
```
### 杀死进程
```shell
kill -9 进程号
```
### 查看端口占用情况
```shell
netstat -lntp
```
- l 表示显示监听状态的端口
- n 表示显示端口对应的 ip 地址和端口号
- t 表示显示 tcp 协议
- p 表示显示进程号和进程名
