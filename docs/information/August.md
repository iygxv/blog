## local-web-server
用于创建本地静态资源服务，支持非常简单的方式设置 proxy(反向代理) 和 SPA 支持

```shell
# 安装
npm install -g local-web-server

# 代理 dist 目录，设置 SPA 入口文件，以 /api 开头请求转发到后端服务
ws -d dist --spa index.html --rewrite '/api/(.*) -> http://127.0.0.1:3000/$1'
```
local-web-server 可以设置代理
