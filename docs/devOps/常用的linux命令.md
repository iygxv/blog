# 常用的linux命令

[Linux 命令大全](https://www.runoob.com/linux/linux-command-manual.html)

后续是我常用的命令

##  查找所有的 nginx.conf 文件
```shell
find . -name nginx.conf
```

## 本地拷贝到远程
```shell
scp -P 22 -r dist/* root@192.168.5.66:/web/xxx
```

## 赋予权限
```shell
# 赋予可执行权限
chmod -x /web/xxx
```
