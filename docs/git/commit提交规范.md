# commit 提交规范

:::tip commit 提交规范

- feat: 新功能
- fix: 修复 bug
- docs: 文档变更
- style: 代码格式（不是指样式发生改变， 是指多了一个；等等的）
- refactor: 重构
- perf: 性能优化
- test: 增加测试
- chore: 构建过程或者辅助工具的变动
- revert: 回退
- build: 打包
  :::
